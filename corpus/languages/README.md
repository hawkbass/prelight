# corpus/languages

One JSON file per language, named by BCP-47 tag (plus special files for emoji, compound-word stress sets).

## File schema (target)

```json
{
  "code": "de",
  "name": "German",
  "direction": "ltr",
  "strings": {
    "save": "Speichern",
    "long_word": "Rechtsschutzversicherungsgesellschaften",
    "sentence_short": "Speichern Sie Ihre Arbeit.",
    "sentence_long": "Wenn Sie diese Nachricht sehen, sollten Sie Ihre Arbeit speichern."
  }
}
```
