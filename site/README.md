# site

The public face of Prelight.

## Two surfaces

- **`index.html`** — landing page. The one-paragraph thesis, three demos with embedded narratives, install snippet. Pretext-style: no marketing fluff, all specifics.
- **`playground.html`** — in-browser playground. The same `@prelight/core` running in the browser, with a live text input, language selector, font scale slider, and width slider. Users *feel* the latency — that's the point.
